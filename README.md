# TappAV - Premium Home Cinema Experience

This is a responsive web application built for showcasing high-end home audio/visual equipment, specifically focusing on the Epson EH-LS12000B Laser Projector.

## Overview

The goal was to create a clean, modern landing page and product page that feels premium. The design uses a deep navy and gold/blue theme to match a "cinema" aesthetic.

## Features

*   **Responsive Design**: Works on mobile and desktop.
*   **Product Showcase**: Detailed specs and imagery for the Epson projector.
*   **Contact Form**: Functional layout for customer inquiries.
*   **Dark Mode UI**: Custom CSS variables for easy theming.

## Tech Stack

*   HTML5
*   CSS3 (Flexbox & Grid)
*   JavaScript (ES6+)

## How to Run

1.  Clone the repository.
2.  Open `index.html` in your browser.
